Privacy Policy

Last Updated: 2025-11-20

Thank you for using this browser extension ("the Extension"). This Privacy Policy explains how the Extension handles matters related to user privacy.

1. General Statement

The Extension does not collect, store, or transmit any personal identifiable information (PII) or usage data. All operations are performed locally within your browser and no data is sent to any remote server.

2. Use of Permissions

If the Extension requests browser permissions (such as activeTab, storage, or scripting), these permissions are used solely to enable local functionality. No user data is collected or transmitted through these permissions.

3. Third-Party Services

The Extension does not use third-party analytics, advertising, or tracking services. No third-party tracking scripts are loaded.

4. Data Retention and Security

Because the Extension does not collect data, there is no stored or transmitted information, and therefore no risk of data leakage. If future versions require collecting or transmitting data, we will update this policy and, if necessary, obtain user consent.

5. Changes to This Policy

If this Privacy Policy undergoes significant changes, the "Last Updated" date will be updated and users will be notified in the extension description or release notes.
